PointBank Privacy Policy

Last Updated: March 2, 2026

1. Data Collected

PointBank stores Discord user IDs and point balances necessary for bot functionality.

2. Data Usage

Collected data is used solely to provide economy features within Discord servers.

3. Data Sharing

No user data is sold, shared, or distributed to third parties.

4. Data Storage

Data is stored securely on a private server and retained only as long as necessary for bot functionality.

5. Contact

For questions regarding privacy, contact the bot administrator.
